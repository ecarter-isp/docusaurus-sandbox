Application

The application service endpoints are used to create and manage client application configurations.

Application - v1

Create application configuration

Creates a new application configuration.

Endpoint:

Method: POST
Type: RAW
URL: {{AppServer}}/app/v1/configs

Headers:

KeyValueDescription
Acceptapplication/json
AuthorizationBearer {{sessiontoken}}Requires the user's session token to authenticate the request.
Content-Typeapplication/json

Body:

{
    "application": "test",
    "settings": {
        "title": "App Title",
        "status": "active",
        "bounds": {
            "top": 5,
            "bottom": -5,
            "left": 2.5,
            "right": -2.5
        }
    },
    "s3bucket": "sloekito-epg-dev-1"
}
